Week 1 - Modelado de Entidades con TypeScript 🏛️
📋 Descripción
Proyecto inicial para aplicar los fundamentos de TypeScript en el modelado de entidades y operaciones básicas de un dominio. El objetivo es practicar interfaces, types, funciones tipadas y estructuras claras.

🎯 Objetivo
Modelar entidades principales de un dominio asignado (ejemplo: biblioteca, farmacia, gimnasio, restaurante).

Usar interfaces y types para definir estructuras.

Implementar funciones con tipos explícitos.

Incluir type unions y type literals.

Documentar el código con comentarios sobre propósito e impacto.

🏗️ Tecnologías
TypeScript

Node.js

pnpm

🚀 Instalación y ejecución
bash
pnpm install
pnpm start
